## หมายเหตุ
- Encoding: UTF-8
- วันที่ใช้รูปแบบ YYYY-MM-DD
- แนะนำให้ import ตามลำดับ: หน่วยนับ, คลัง, ตำแหน่งจัดเก็บ, กลุ่มลูกค้า/ผู้ขาย, เงื่อนไขชำระ, ภาษี, สินค้า, คู่ค้า, และ BOM
